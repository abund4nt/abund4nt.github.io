We have the values $p$, $q$, $dp$, $dq$, and $c$. At first glance we can see that we are facing a cryptosystem similar to RSA. Doing some research on Google I came across the following post where he explains that there is a variant called RSA-CRT, which uses the Chinese remainder theorem to encrypt data values that we have were calculator using the following mathematical operations.

$$ \begin{equation*} dp = d \text{ (mod } p - 1 \text{)} \end{equation*} $$ $$ \begin{equation*} dq = d \text{ (mod } q - 1 \text{)} \end{equation*} $$

In order to break this algorithm we need to calculate $q_{inv}$, performing this mathematical operation is obtained.

$$ \begin{equation*} q_{inv} = q^{-1} \text{ (mod } p \text{)} \end{equation*} $$

But in this case it is not necessary to obtain $q_{inv}$, since we have all the necessary values to decrypt the message since:

$$ \begin{equation*} m = c^{d_{q}} \text{ (mod } q \text{)} \end{equation*} $$
