Using the following Python script we will determine if both public keys are the same, and if they differ in their $n$ or $d$ exponent.

from Crypto.PublicKey import RSA

key1 = RSA.import_key(open('key1.pem').read())
key2 = RSA.import_key(open('key2.pem').read())

print(key1.n == key2.n)
print(key1.e == key2.e)

We use the RSA module of the Crypto.PublicKey library and print on the screen the result of both equalities, when the script is executed it will return a boolean value.

➜  TwoForOne python3 verify_keys.py
True # n
False # e

We see that the n values are the same but the exponent $e$ is different, as both $n$ values are the same we can determine that both messages have the same content but are encrypted with different exponent.

RSA Context

RSA encrypts a message in decimal format $m$ we can encrypt it with the following equation.

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

The public key is formed by $n$ and $e$. $n = p \cdot q$ which are two large prime numbers multiplied (these are stored as the private key).

In order to decrypt the message you need the value $\phi$($n$) $=$ ($p - 1$) $\cdot$ ($q - 1$). Then.

$$ \begin{equation*} m = c^{d} \text{ (mod } n \text{)} \end{equation*} $$

Common module attack

To solve this challenge we have to make a common module attack, doing some research I came across the following paper that explains it well. Although a simple Google search is enough to understand how it works.

A Common Modulus attack can be used to recover the plaintext when the same message is encrypted to two RSA keys that use the same modulus. This algorithm works if and only if message sends with the same modulus and relatively prime encryption exponents.

We have two nonzero integers, this means that there exist integers $x$ and $y$, giving us the possibility to apply the bezout identity. The bezout identity is an elementary theorem of number theory that explains that if a and $b$ are nonzero integers with $gcd(d)$ then there exist integers $x$ and $e$ such that:

$$ \begin{equation*} ax+by = d \equiv ax+by = gcd(d) \end{equation*} $$
