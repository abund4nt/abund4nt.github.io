Se nos proporciona una instancia remota para conectarnos y el codigo fuente del servidor server.py.

from secret import FLAG, N

MENU = '''
YALM (Yet Another Lost Modulus):

1. Get secret
2. Test encryption
3. Exit
'''


class YALM:

    def __init__(self):
        self.e = 3
        self.n = N

    def get_secret(self):
        message = f'Hey! This is my secret... it is secure because RSA is extremely strong and very hard to break... Here you go: {FLAG}'
        m = int(message.encode().hex(), 16)
        c = pow(m, self.e, self.n)

        return hex(c)

    def test_encryption(self):
        plaintext = input('Plaintext: ').strip()

        m = int(plaintext, 16)
        cs = []

        while m:
            c = pow(m, self.e, self.n)
            cs.append(c)
            m //= self.n

        if len(cs) > 1:
            return 'Too many messages!'

        return 'Thanks for the message!'


def main():
    yalm = YALM()

    while True:
        print(MENU)

        option = int(input('Option: '))

        if option == 1:
            print(f'Ciphertext: {yalm.get_secret()}')
        elif option == 2:
            print(yalm.test_encryption())
        else:
            break


if __name__ == '__main__':
    main()

Analisis del codigo fuente

En el script se importa la flag y \(n\) luego define el exponente publico \(e = 3\). Tenemos dos funciones relevantes, la primera es get_secret().

def get_secret(self):
        message = f'Hey! This is my secret... it is secure because RSA is extremely strong and very hard to break... Here you go: {FLAG}'
        m = int(message.encode().hex(), 16)
        c = pow(m, self.e, self.n)

        return hex(c)

En esta funcion tenemos un mensaje el cual contiene la flag, luego el mensaje lo convierte en hexadecimal y finalmente lo encripta utilizando el exponente publico \(e\) y \(n\).

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

La otra funcion relevante es.

def test_encryption(self):

        m = int(plaintext, 16)
        cs = []

Esta funcion tiene un input que almacena en la varible plaintext, luego lo convierte a hexadecimal y define un arreglo vacio llamado cs. Mientras m exista este sera encriptado utilizando el exponente publico \(e\) modulo \(n\), añadira el texto cifrado al arreglo, si todo sale bien nos muestra el Thanks for the message!. Si \(m > n\) nos mostrara el error Too many messages!. En resumen estamos ante un criptosistema RSA.

RSA Context.

RSA encrypts a message in decimal format \(m\) we can encrypt it with the following equation.

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

The public key is formed by \(n\) and \(e\). \(n = p \cdot q\) which are two large prime numbers multiplied (these are stored as the private key).

In order to decrypt the message you need the value \(\phi\)(\(n\)) \(=\) (\(p - 1\)) \( \cdot \) (\(q - 1\)). Then

$$ \begin{equation*} m = c^{d} \text{ (mod } n \text{)} \end{equation*} $$

Recuperacion del modulo

Si leemos la descripcion del challenge podemos ver que nos piden recuperar el modulo perdido (\(n\)).

I created an encryption server with RSA, but I forgot to show the modulus. Can you help me recover it yet another time?

Sabiendo que si \(m > n\) el servidor nos respondera con un error podemos determinar la longitud de \(n\), al saber esto podemos aplicar el algoritmo de Busqueda Binaria para recuperar \(n\). Utilizando el siguiente script recuperamos el modulo.

from pwn import *

def get_process():
    if len(sys.argv) != 2:
        print('Usage: python3 solve.py <ip:port>')
        exit(1)
    host, port = sys.argv[1].split(':')
    return remote(host, int(port))

def main():

    conn = get_process()

    high = pow(2, 2048) # len = 617

    low = 0
    mid = 0

    while low <= high:
        mid = (high + low) // 2

        # Pwntools
        conn.sendlineafter(b'Option: ', b'2')
        conn.recvuntil(b"Plaintext: ")
        conn.sendline(hex(mid))
        line = conn.recvline()
   
        if b"Too many messages!" in line:
            high = mid - 1
        else:
            low = mid + 1
    
        print("N = ", hex(low))

if __name__ == '__main__':
    main()
