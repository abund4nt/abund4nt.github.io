We have the flag encrypted with AES, a key in hexadecimal format and the public key. With the public key we can get the values of $n$ and $e$.

RSA Context

RSA encrypts a message in decimal format $m$ we can encrypt it with the following equation.

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

The public key is formed by $n$ and $e$. $n = p \cdot q$ which are two large prime numbers multiplied (these are stored as the private key).

In order to decrypt the message you need the value $\phi$($n$) $=$ ($p - 1$) $\cdot$ ($q - 1$). Then.

$$ \begin{equation*} m = c^{d} \text{ (mod } n \text{)} \end{equation*} $$

Values $n$ and $e$.

With the public key we can get the $n$ and $e$ values, we have to use the RSA module of the Crypto.PublicKey library, we will use the following Python script.

from Crypto.PublicKey import RSA
with open('pubkey.pem') as f:
    key = RSA.import_key(f.read())

print(key.e)
print(key.n)

We use Python’s open() function to read the pubkey.pem file and we can extract $n$ and $e$.

Something strange is happening here since $n = p^{3}$, this means that we are not dealing with a common RSA system, the good thing is that this is not an impediment to progress since we can calculate $\phi$ as $p^{3} - p^{2}$ (https://es.wikipedia.org/wiki/Función_φ_de_Euler).

As the flag was encrypted with AES, the first thing I did was to decrypt the key, this was achieved with the following script.

from Crypto.PublicKey import RSA
from Crypto.Util.number import * 

with open('pubkey.pem') as f:
    key = RSA.import_key(f.read())

with open('key') as f:
    c = int(f.read(), 16)

p = 10410080216253956216713537817182443360779235033823514652866757961082890116671874771565125457104853470727423173827404139905383330210096904014560996952285911
phi_n = p ** 3 - p ** 2
d = inverse(key.e, phi_n)
m = pow(c, d, key.n)
k = bytes.fromhex(hex(m)[2:])

print(k)

With this we can decrypt the flag.txt.aes file and obtain the flag.

Flag

We add the following lines of code to read the flag.

from Crypto.Cipher import AES
cipher = AES.new(k, AES.MODE_ECB)

with open('flag.txt.aes', 'rb') as f:
    ct = f.read().strip()

flag = cipher.decrypt(ct)

print(flag)

Complete script.

from Crypto.PublicKey import RSA
from Crypto.Util.number import * 
from Crypto.Cipher import AES

with open('pubkey.pem') as f:
    key = RSA.import_key(f.read())

with open('key') as f:
    c = int(f.read(), 16)

p = 10410080216253956216713537817182443360779235033823514652866757961082890116671874771565125457104853470727423173827404139905383330210096904014560996952285911
phi_n = p ** 3 - p ** 2
d = inverse(key.e, phi_n)
m = pow(c, d, key.n)
k = bytes.fromhex(hex(m)[2:])

cipher = AES.new(k, AES.MODE_ECB)

with open('flag.txt.aes', 'rb') as f:
    ct = f.read().strip()

flag = cipher.decrypt(ct)

print(flag)
