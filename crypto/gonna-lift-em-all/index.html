Source code analysis

This is a cryptosystem that uses modular exponentiation operations to perform encryption and key generation. $c1$ and $c2$ are the encrypted messages that are generated as follows.

$$ \begin{equation*} C1 = g \cdot y\text{ (mod } n \text{)} \end{equation*} $$ $$ \begin{equation*} C2 = m \cdot s\text{ (mod } n \text{)} \end{equation*} $$

The vulnerability of this cipher lies in the output since it gives us a lot of information ($p$, $q$, $h$, $c1$ and $c2$). We must find $y$ and $s$, to find these values we must solve the following operations.

$$ \begin{equation*} y = (c1 \cdot g^{-1})\text{ mod } p \text{} \end{equation*} $$ $$ \begin{equation*} s = (h^{y}) \text{ mod } p \text{} \end{equation*} $$
