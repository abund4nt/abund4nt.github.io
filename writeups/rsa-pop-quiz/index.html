We are facing a surprise questionnaire about RSA, in this case it asks us if it is possible to calculate \(n\) using \(p\) and \(q\).

RSA Context.

RSA encrypts a message in decimal format $m$ we can encrypt it with the following equation.

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

The public key is formed by \(n\) and \(e\). \(n = p \dot q\) which are two large prime numbers multiplied (these are stored as the private key).

In order to decrypt the message you need the value \(\phi\)(\(n\)) \(=\) (\(p - 1\)) \( \dot \) (\(q - 1\)). Then

$$ \begin{equation*} m = c^{d} \text{ (mod } n \text{)} \end{equation*} $$

For a Python implementation of RSA you can look at this:

https://github.com/luk4spwn/rsa-implementation/

IS THIS POSSIBLE and FEASIBLE? (Y/N):

The Euler totient function is the number of positive integers less than or equal to a given integer that are relative primes with respect to it. Recall that \(\phi\) is calculated \((p - 1) * (q - 1)\). Using interactive Python we can easily derive this value.

>>> p = 12611
>>> q = 66347
>>> totient = (p - 1) * (q - 1)
>>> totient
836623060

Or also using sympy’s totient module. Program the following script and we will get the same result as above.

from sympy.ntheory.factor_ import totient

n = 836702017

totient_n = totient(n)

print(totient_n)
# 836623060

We have the necessary values to calculate $d$. Using the following Python script we can calculate it.

def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

def extended_gcd(a, b):
    if a == 0:
        return (b, 0, 1)
    else:
        gcd, x, y = extended_gcd(b % a, a)
        return (gcd, y - (b // a) * x, x)

def mod_inverse(a, m):
    gcd, x, y = extended_gcd(a, m)
    if gcd != 1:
        raise Exception('No existe inverso modular')
    else:
        return x % m

def calculate_d(p, q, e):
    phi = (p - 1) * (q - 1)
    d = mod_inverse(e, phi)
    return d

p = int(input("Ingrese el valor de p: "))
q = int(input("Ingrese el valor de q: "))
e = int(input("Ingrese el valor de e: "))
