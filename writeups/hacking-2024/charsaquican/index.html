Todos los ingredientes están mezclados! Es un charquicán.

Solution

To solve this challenge it is enough to understand the description of the challenge, the values given correspond to the parameters of the RSA cryptosystem ($p$, $q$, $e$ and $n$).

After a while playing with the values I was able to determine that the first argument (0) corresponds to $p$ and the last argument $3$ corresponds to $q$, with the Python console I was able to calculate $n$ since $n$ = $p$ or $q$.

$ python3 -q

Knowing the value of $p$ and $q$ we can calculate $\phi$ which is equal to $(p - 1) \cdot (q - 1)$ , with this value we can calculate $d$ and obtain the flag.
