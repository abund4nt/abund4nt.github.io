The purpose of this challenge is to break \(c\) to find the flag, we see that we are facing the RSA cryptosystem.

Brief explanation of RSA.

RSA encrypts a message in decimal format $m$ we can encrypt it with the following equation.

$$ \begin{equation*} c = m^{e} \text{ (mod } n \text{)} \end{equation*} $$

The public key is formed by \(n\) and \(e\). \(n = p \cdot q\) which are two large prime numbers multiplied (these are stored as the private key).

In order to decrypt the message you need the value \(\phi\)(\(n\)) \(=\) (\(p-1\)) \(\cdot\) (\(q - 1\)). Then

$$ \begin{equation*} m = c^{d} \text{ (mod } n \text{)} \end{equation*} $$

For a Python implementation of RSA you can look at this:

https://github.com/abund4nt/rsa-implementation

Solution

When $e$ is very large or has the same length as \(n\) we can perform the Wiener attack to recover \(d\), there is a module in Python that allows us to perform this attack called owiener, using the following script we can obtain \(d\).

e = 38992200990947084444616822481761525205200564263400548947646537584390391307631989872285937907556276689558785070126811542318404206151575929566856619052256673251700199716234526453215467641158671154963660640312471138156646420619850127528032342928039653079176790826202649972056973495042955152930392869564972388205
