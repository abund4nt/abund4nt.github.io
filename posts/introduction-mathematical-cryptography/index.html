Divisibilidad

Sean $a$ y $b$ numeros enteros con $b \neq 0$. Se dice que $b$ divide $a$ si existe un entero $c$ que.

$$a = bc$$

Se escribe $b \mid a$ para indicar que $b$ divide $a$. Si $b$ no divide $a$ entonces se escribe $b \nmid a$.

Ejemplo: Tenemos $847 \mid 485331$ por que $485331 = 847 \cdot 573$. Por otro lado tenemos $355 \nmid 259943$ por que si intentamos dividir $259943$ en $355$ obtenemos un resto de $83$, para ser precisos $259943 = 355 \cdot 732 + 83$, entonces $259943$ no es un multiplco exacto de $355$.

Maximo comun divisor

El maximo comun divisor entre $a$ y $b$ es el entero positivo mas largo $d$ que $d \mid a$ y $d \mid b$, esto se denota como $gcd(a, b)$.

Ejemplo: El maximo comun divisor entre $12$ y $18$ es $6$, por que $6 \mid 12$ y $6 \mid 18$, tambien.

$$gcd(748, 2024) = 44$$

Lista de los divisores de 748 y 2024.

$$\text{Divisores de 748 }= 1, 2, 4, 11, 17, 22, 34, 44, 68, 187, 374, 74 $$ $$\text{Divisores de 2024 }= 1, 2, 4, 8, 11, 22, 23, 44, 46, 88, 92, 184, 253, 506, 1012, 2024$$

Un algoritmo eficiente para calcular el maximo comun divisor entre dos numeros es el algoritmo de euclides, esta es su implementacion en Python.

def gcd(a, b):
    if a == 0:
        return b
    return gcd(b % a, a)
